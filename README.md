## Activating GitHub Pages

These are static HTML files; GitHub Pages is not yet enabled on this repo.
To get clean `https://pheobo12.github.io/app-legal/...` URLs:

1. Go to **Settings → Pages** in this repository
2. Under "Build and deployment", set **Source: Deploy from a branch**
3. Branch: `main`, folder: `/ (root)` → **Save**

Until Pages is enabled, the raw file URLs
(`https://raw.githubusercontent.com/Pheobo12/app-legal/main/privacy.html`)
work but render as plain text rather than a formatted page.
